# NG Network Demo Website

React JS + Tailwind CSS + JavaScript based premium responsive landing page.

## Included sections
- Hero section with background video support
- AE Impact scrolling section
- About NG Network
- Admission section
- Testimonials
- YouTube video scrolling section
- About Nitin Goswami

## Run locally
```bash
npm install
npm run dev
```

## Important edits
- Add hero video at: `public/videos/hero-bg.mp4`
- Update text/data at: `src/data/siteData.js`
- Replace logo/images if needed in `public/images/`

## Tech used
- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
